# CAT Exam Rules

## Sectional Structure

The CAT exam follows a **fixed, non-reversible order**:

1. VARC (Verbal Ability & Reading Comprehension)
2. DILR (Data Interpretation & Logical Reasoning)
3. QA (Quantitative Ability)

## Time Limits

- Each section is allotted **40 minutes**
- Total exam duration: **120 minutes**
- Once a section's time ends, you **cannot** go back to review or answer questions in that section

## Navigation Rules

- You **cannot** move between sections during the exam
- You can only navigate between questions **within the current section**
- Candidates must focus on the current section only — time management is crucial

## Scoring

| Result | Marks |
|--------|-------|
| Correct answer | +3 |
| Incorrect answer | −1 |
| Unattempted | 0 |
